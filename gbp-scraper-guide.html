📋 Step-by-step guide

GBP Review Scraper

Install the Chrome extension and export Google Business reviews to a CSV file — no technical experience needed.

Step 1 — Download the extension Click the button to download the zip file to your computer.
⬇ Download ZIP
Part 1 — Installation
1
Download the extension file ▼
Get the zip file containing the extension.

Click the Download ZIP button at the top of this page. Save the file somewhere easy to find — your Desktop or Downloads folder is ideal.

💡 Once downloaded, leave the file where it is for now — the next step will walk you through unzipping it.
2
Unzip the file ▼
Extract the folder from the zip file.

Find the downloaded file and unzip it:

On Windows Right-click gbp-review-scraper.zip → Extract All → click Extract
On Mac Double-click gbp-review-scraper.zip — it extracts automatically
💡 You'll end up with a folder called gbp-scraper. Keep this folder somewhere permanent — Chrome needs to be able to find it every time.
3
Open Chrome's extensions page ▼
Navigate to where Chrome manages extensions.

In Chrome, click the address bar at the top, type the address below exactly, and press Enter:

chrome://extensions
The Extensions management page will open here.
4
Turn on Developer mode ▼
Required to load extensions from your own computer.

On the extensions page, look for Developer mode in the top-right corner. Click the toggle to switch it on — it turns blue when active.

Developer mode
⚠️ Don't worry — this is safe. Developer mode simply allows you to load extensions from your computer instead of only the Chrome Web Store.
5
Load the extension ▼
Point Chrome to the folder you unzipped.

After turning on Developer mode, three new buttons appear at the top of the page. Click Load unpacked:

Load unpacked
Pack extension
Update

A file browser window will open. Navigate to the gbp-scraper folder (the one you unzipped) and select it. Then click Select Folder (Windows) or Open (Mac).

💡 Select the folder itself, not a file inside it. The correct folder contains files named manifest.json, content.js, and others.
6
Pin the extension to your toolbar ▼
So the icon is always visible and easy to click.

The extension is now installed. To make the icon visible in your toolbar at all times:

  1. Click the puzzle piece icon 🧩 in the top-right corner of Chrome
  2. Find GBP Review Scraper in the list
  3. Click the pin icon 📌 next to it

The extension icon will now appear in your Chrome toolbar on every page.

Part 2 — Using the extension
7
Go to a Google Business listing ▼
Find the business you want to scrape reviews for.

The extension works on two types of Google pages:

Option A — Google Search Search for a business (e.g. Costa Coffee Manchester) — the business info panel with reviews should appear on the right side of the results page.
Option B — Google Maps Go to maps.google.com, search for the business and click on it to open the listing in the left sidebar.
⚠️ Make sure the page has fully loaded before clicking the extension icon. Wait until the reviews are visible on screen.
8
Click the extension icon ▼
Start the scrape with a single click.

Click the GBP Review Scraper icon in your Chrome toolbar. A panel will open on the right side of the screen and scraping begins automatically — you don't need to do anything else.

💡 The extension handles everything automatically: clicking the Reviews tab, sorting by newest, scrolling to load all reviews, and expanding any truncated text.
9
Wait for it to finish ▼
Watch the progress bar in the side panel.

The side panel shows a progress bar and five steps. Each step lights up as the extension completes it:

✓
Reviews tab
✓
Sort
3
Load all
4
Expand
5
Extract
⏱️ A business with 50 reviews takes roughly 30 seconds. 500+ reviews may take 2–3 minutes. Don't switch away from the tab while it's running.
10
Download the CSV file ▼
Save your reviews as a spreadsheet-ready file.

When scraping finishes, a preview table of reviews appears in the side panel and a blue Download CSV button appears at the bottom. Click it to save the file to your Downloads folder.

The CSV file can be opened directly in:

  1. Microsoft Excel — double-click the file to open
  2. Google Sheets — go to sheets.google.com → File → Import → Upload the CSV
  3. Numbers (Mac) — double-click the file to open
📊 The file contains 5 columns: reviewer name, star rating, date, review text, and any owner response.
✅

You're all set!

The extension is installed and ready to use. Navigate to any Google Business listing on Google Search or Google Maps and click the icon to start scraping reviews.
